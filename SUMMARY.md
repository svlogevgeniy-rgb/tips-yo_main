## Как использовать

### Сейчас:
```bash
# Сервер уже запущен на:
http://localhost:5173
```

### Добавить изображения:
1. Переименуйте 6 PNG-файлов согласно инструкции
2. Скопируйте в `public/images/`
3. Обновите страницу в браузере

### Сборка для продакшена:
```bash
npm run build
```
