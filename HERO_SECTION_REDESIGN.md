- Фокус на простоте процесса

```tsx
<img src="/images/Logo.svg" />  // Логотип
<h1>Tips'yo</h1>                // Название бренда

<svg>                           // Иконки trust indicators
```

### Fallback для логотипа
```tsx
onError={(e) => {
  const target = e.target as HTMLImageElement;
  target.src = '/images/Logo.png';
}}
```

Если SVG не загрузится, автоматически подставится PNG версия.
