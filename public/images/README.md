# Изображения для карточек профессий

Поместите следующие изображения в эту папку:

1. **waitress.png** — Официанты (бариста/официантка с кофе)
2. **manicure.png** — Мастера маникюра (мастер за работой)
3. **blogger.png** — Блогеры (создание контента)
4. **gas-station.png** — Заправщики (ночная заправка)
5. **courier.png** — Курьеры (курьер с рюкзаком)
6. **anyone.png** — Кто угодно (гости с покупками)

## Формат изображений

- Формат: PNG
- Рекомендуемый размер: 400×600px (портретная ориентация)
- Соотношение сторон: 2:3

## Как добавить изображения

1. Сохраните предоставленные PNG-файлы в эту папку.
2. Убедитесь, что имена файлов соответствуют указанным выше.
3. Перезагрузите страницу в браузере.

Если изображения не найдены, карточки автоматически используют градиентный фон как fallback.
