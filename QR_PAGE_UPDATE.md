4. **Кнопки действий** (только на mobile)
   - "Скачать QR-код"
   - "Поделиться"

#### Правая колонка - QR-код:
1. **Карточка с QR-кодом**
   - Большой квадратный QR-код
   - Белый фон с лёгкой рамкой

2. **Информация о QR**
   - ID сотрудника

3. **Кнопки действий** (только на desktop)

4. **Дополнительная информация**

1. **Генерация реального QR-кода**
   - Установить библиотеку: `npm install qrcode.react`
