# 🎨 Объединение Header и Hero-секции

## Обзор изменений

Header и Hero-секция теперь представляют собой единый цельный блок с общим зелёным фоном. Логотип и название перенесены из центра секции в левый верхний угол header.

---

## 🎯 Ключевые изменения

### 1. Логотип и название → в Header
- **Было:** Логотип и "Tips'yo" по центру hero-секции
- **Стало:** Логотип и "Tips'yo" слева в header

### 2. Header → зелёный фон
- **Было:** Белый фон (`bg-white`)
- **Стало:** Зелёный фон (`bg-primary` = #0C4A3D)

### 3. Hero-секция → без логотипа
- **Было:** Логотип + название + контент
- **Стало:** Только контент (заголовок, описание, шаги, CTA)

---

## 📐 Обновлённая структура Header

### Композиция

```
┌─────────────────────────────────────────────────────────────┐
│  [Logo] Tips'yo    Для кого  Как работает  Сколько  [CTA]  │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### Элементы слева направо:

**1. Логотип + Название (слева)**
```tsx
<div className="flex items-center gap-3">
  <img src="/images/Logo.svg" className="h-8 md:h-10" />
  <span className="text-xl md:text-2xl font-bold text-white">
    Tips'yo
  </span>
</div>
```

**Характеристики:**
- Layout: `flex items-center gap-3`
- Логотип: h-8 (32px) на mobile, h-10 (40px) на desktop
- Название: text-xl (20px) на mobile, text-2xl (24px) на desktop
- Цвет текста: `text-white`
- Вес шрифта: `font-bold`

**2. Навигация (центр)**
```tsx
<nav className="hidden md:flex items-center space-x-8">
  <button className="text-white/90 hover:text-white">
    Для кого
  </button>
  {/* ... */}
</nav>
```

**Характеристики:**
- Скрыта на mobile: `hidden md:flex`
- Отступы между пунктами: `space-x-8` (32px)
- Цвет: `text-white/90` (90% прозрачность)
- Hover: `hover:text-white` (100% белый)
- Вес: `font-medium`

**3. CTA кнопки (справа)**
```tsx
<div className="hidden md:flex items-center space-x-4">
  <Button variant="outline">Поблагодарить</Button>
  <Button>Начать пользоваться</Button>
</div>
```

**Характеристики:**
- Скрыты на mobile: `hidden md:flex`
- Отступ между кнопками: `space-x-4` (16px)

**Кнопка 1 (Outline):**
- Граница: `border-2 border-white`
- Текст: `text-white`
- Hover: `hover:bg-white hover:text-primary`

**Кнопка 2 (Primary):**
- Фон: `bg-white`
- Текст: `text-primary` (зелёный)
- Hover: `hover:bg-white/90`

**4. Мобильное меню (справа на mobile)**
```tsx
<button className="md:hidden">
  <Menu className="h-6 w-6 text-white" />
</button>
```

## 🎭 Обновлённая композиция Hero-секции

### Структура (без логотипа)

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│              Благодарность, подарки                          │
│              и чаевые безналично                             │
│                                                              │
│         Простой и безопасный способ...                       │
│         Сканируйте QR-код...                                 │
│                                                              │
│         [1] Сканируйте  [2] Выберите  [3] Оплатите          │
│                                                              │
│         [Подключить заведение] [Оставить чаевые]             │
│                                                              │
│         🛡️ Безопасные • 💰 0% • ⚡ Мгновенно                    │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### Визуальный фокус

**Главный элемент:** Заголовок H2
```
Благодарность, подарки
и чаевые безналично
```

**Иерархия внимания:**
1. 🎯 Заголовок (самый крупный, жирный)
2. 📝 Описание (средний размер, 90% прозрачность)
3. 🔢 Шаги процесса (визуальные элементы)
4. 🔘 CTA кнопки (яркие, контрастные)
5. ✅ Trust indicators (мелкие, внизу)

### Отступы

**Секция:**
- Top: `pt-24` (96px) на mobile, `pt-32` (128px) на desktop
- Bottom: `pb-24` (96px) на mobile, `pb-32` (128px) на desktop
- Horizontal: `px-4` (16px)

**Между элементами:**
- Заголовок → Описание: `mb-6` (24px)
- Описание → Шаги: `mb-8` (32px)
- Шаги → Кнопки: `mb-10` (40px)
- Кнопки → Trust indicators: `mb-6` (24px)

### Выравнивание

- Всё по центру: `text-center`
- Container: `max-w-3xl mx-auto`
- Шаги на desktop: центрированы
- Шаги на mobile: слева

## 🔄 Сравнение: До и После

### До (разделённые блоки)

```
┌─────────────────────────────────────┐
│ [Logo] tipsyo    Menu    [Buttons]  │ ← Белый header
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│         [Logo.svg]                  │
│          Tips'yo                    │ ← Зелёная hero
│                                     │
│    Благодарность, подарки...        │
└─────────────────────────────────────┘
```

**Проблемы:**
- ❌ Визуальный разрыв между header и hero
- ❌ Дублирование логотипа/названия
- ❌ Разные цвета (белый vs зелёный)
- ❌ Нет единства

### После (единый блок)

```
┌─────────────────────────────────────┐
│ [Logo] Tips'yo  Menu    [Buttons]   │ ← Зелёный header
├─────────────────────────────────────┤
│                                     │
│    Благодарность, подарки...        │ ← Зелёная hero
│                                     │
│         (контент)                   │
│                                     │
└─────────────────────────────────────┘
```

**Улучшения:**
- ✅ Единый зелёный блок
- ✅ Логотип только в header
- ✅ Визуальная целостность
- ✅ Профессиональный вид
- ✅ Лучшая иерархия
