# Troubleshooting

Расширенная версия блока из `LANDING_READY.md` с типовыми проблемами.

## Стили не применяются

1. Остановите dev-сервер (`Ctrl + C`) и перезапустите `npm run dev`.
2. Очистите кеш браузера (`Cmd + Shift + R`).
3. Убедитесь, что `src/index.css` импортирован в `src/main.tsx`.

## Порт занят

- Vite предложит свободный порт (5174+). При желании указать порт вручную запустите `npm run dev -- --port=5174`.

## Не показываются изображения профессий

- Проверьте имена файлов в `public/images`.
- Удалите кеш браузера: компоненты могут запоминать отсутствующее изображение.
- Посмотрите вывод `npm run dev`: ошибки загрузки видны в терминале и DevTools.

## Ошибки при сборке

- Запустите `npm run lint` — многие ошибки TypeScript/Tailwind обнаруживаются заранее.
- Убедитесь, что используется Node.js 18+. Старые версии вызывают ошибки Vite/ESBuild.

## Медленная загрузка после деплоя

- В production используйте статический хостинг с gzip/brotli.
- Удалите тяжёлые изображения: оптимизируйте PNG до <500 KB.

## Интерактивность не работает

- Проверьте, не удалены ли обработчики `onClick` в секциях CTA.
- Если используется внешняя аналитика/чат, убедитесь, что токены прописаны в `.env`.
