# Accessibility Guide

## Контраст
- Минимум 4.5:1 для текста, 3:1 для крупного текста/иконок.
- Проверяем primary на dark/light фонах (см. Figma tokens).

## Клавиатура
- Все интерактивные элементы — `button`, `a` или `input`. Неиспользуем `div role=button` без причины.
- Порядок фокуса соответствует визуальному.
- `Button`, `Input`, `AmountPill` прокидывают `...rest` и `aria-*` атрибуты.

## ARIA / Live regions
- Ошибки форм связываем через `aria-describedby`.
- Toast'ы помечаем `role="status"` или `alert` в зависимости от критичности.

## Motion / reduced motion
- Добавить хук `usePrefersReducedMotion` (todo) и отключать анимации.

## Состояния сообщений
- Цвета ошибок/успехов сопровождаем иконками/текстом.
- Placeholder не заменяет label.

## Международные пользователи
- Все тексты локализуем (i18n), формат валют/дат через Intl.
