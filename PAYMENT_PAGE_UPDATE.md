## 💡 Интерактивность

### 1. Выбор предустановленной суммы
```typescript
const handleAmountSelect = (amount: number) => {
  setSelectedAmount(amount)
  setCustomAmount('') // Сбрасываем кастомную сумму
}
```

**Поведение:**
- Клик на кнопку → подсветка зелёным
- Автоматически сбрасывается кастомная сумма
- Показывается итоговая сумма

### 2. Ввод кастомной суммы
```typescript
const handleCustomAmountChange = (value: string) => {
  setCustomAmount(value)
  setSelectedAmount(null) // Сбрасываем выбор кнопки
}
```

**Поведение:**
- Ввод в поле → сброс выбора кнопок
- Автоматический расчёт итоговой суммы
- Валидация (только числа)

### 3. Финальная сумма
```typescript
const finalAmount = customAmount ? parseInt(customAmount) : selectedAmount
```

**Логика:**
- Приоритет у кастомной суммы
- Если кастомная пустая → используется выбранная кнопка
- Используется для расчётов и кнопки оплаты

### 4. Кнопка оплаты
```tsx
<Button
  disabled={!finalAmount || finalAmount <= 0}
>
  Оплатить {finalAmount > 0 ? `${finalAmount} ₽` : ''}
</Button>
```

**Состояния:**
- Disabled: сумма не выбрана или ≤ 0
- Active: показывает сумму в тексте кнопки
